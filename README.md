# Flappy Bird

This is a clone of the Flappy Bird game made using the pygame module in python.

# Installation

1. Download the code in a ZIP format and unzip it in a folder of your choice.
2. Open Terminal and install pygame modules.

```bash
  pip install pygame
  python3 -u main.py
```
